Ejercicios de repaso js

  • Crear un Crud usando una clase para los tres metodos de creación, actualización y eliminación, usar bootstrap, utiliza delegación de eventos
  • Dado el arreglo "Data" de objetos crea una barra de busqueda que muestre los usuarios encontrados conforme vayan escribiendo y dando enter en el input
  • Crear un minijuego donde el objeto salte cada vez que le pasemos el mouse por encima y cuente los puntos realizados
  • Cree una minilibrería llamada jjs que contenga 3 nuevas formas de seleccionar elementos, id => jid, name => jn, className => jc, y que tenga una función llamada crear header, que nos va a pedir de parametro un titulo, el color del titulo, una imagen y un texto para un botón, contenedor padre
